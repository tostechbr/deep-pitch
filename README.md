# ⚽ deep-pitch

**Deep Agent que prevê os mata-matas da Copa do Mundo 2026** — reconciliando um
modelo estatístico (Dixon-Coles / Elo), dados ao vivo e contexto qualitativo, e
explicando *o porquê* de cada previsão.

Não é "mais um modelo de xG". É um agente que raciocina como analista: parte de
um prior estatístico, ajusta com o estado atual do torneio (lesões, forma,
escalação) e se compara a benchmarks (Opta Supercomputer, Hicruben).

Construído com **Deep Agents** (LangChain), **model-agnostic** (Claude, Gemini,
Groq, OpenRouter, NVIDIA), observável via **LangSmith**.
